## 🎓 LESSONS LEARNED & BEST PRACTICES

### Architecture Patterns

**Dual-Mode System:**
- Separate color palettes for distinct user flows
- Reusable UI components with color props
- Mode-specific hooks for feature detection
- Consistent design language across both modes

**Component Organization:**
- Feature-based directory structure
- Shared components in common directory
- Clear separation of concerns
- Reusable utility functions

**State Management:**
- React hooks for component state
- Zustand for global state (cart)
- Supabase for server state
- Proper dependency management

### Performance Optimization

**Frontend:**
- Image optimization with Next.js Image
- Code splitting and lazy loading
- CSS-in-JS with Tailwind
- Memoization of expensive components

**Backend:**
- Edge functions for near-instant response
- Database query optimization
- Pagination for large datasets
- Caching strategies

### Security Best Practices

- Environment variables for secrets
- Row-level security on database
- API rate limiting
- HTTPS enforcement
- Input validation and sanitization
